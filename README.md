# Официальный обработчик Robokassa для 1С УТ, КА – многопользовательская версия

### Инструкция по использованию
Данный проект представляет собой инструмент, оформленный в виде 1С‑расширения, который позволяет работать с конфигурациями УТ, КА, ЕРП, БП и УНФ. Он автоматически подстраивается под нужные настройки, а порядок использования также описан в [инструкции](https://docs.robokassa.ru/media/guides/Robokassa_1c_prepriyatie.pdf).

### Установка и настройка
1. Установите расширение Робокасса и отключите безопасный режим (при установке потребуется реструктуризация базы данных, рекомендуется предварительно выполнить резервное копирование, а также потребуется монопольный доступ).
2. Если обработка «Робокасса» ранее уже использовалась, после установки один раз выполните обработку «Обработка переноса данных (Робокасса)», которую можно найти через меню «Функции для технического специалиста».
3. При необходимости настройте расписание регламентного задания «Обновление статусов счетов» (по умолчанию выполняется каждые 5 минут), отвечающего за частоту обновления статусов по выставленным счетам.

### Открытие инструмента
Инструмент можно открыть из документов продаж, либо запускать его как внешний файл, в зависимости от сценария работы.

### Дополнительная информация
Данная версия обработки поддерживает одновременную работу менеджеров на множестве компьютеров, что удобно для многопользовательских инсталляций.  
Для более подробной информации о возможностях и настройках инструмента следует обращаться к соответствующей документации, а перед использованием убедиться в актуальности версии и корректности выполненных настроек.
